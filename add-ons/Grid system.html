Yjsg v2 is equipped with mobile first, responsive grid system layout that can be used in any Joomla pages extensions or modules.
To take advnatage of this grid system just add yjsg-row container and inside you can have up to 8 column blocks.
The examples below display the usage that is suitable for any page layout.

markup:

[yjsgpre pretty="1" scroll="0"]
content...
content...
[/yjsgpre]
